Instructions for Adding a User to the Stack Overflow Moderation Election Visualization

Step 1: Query For Information from Main Site

Please run the linked SEDE query, entering your Stack Overflow user ID (the number in the URL when you go to your user profile on the main Stack Overflow site), your current helpful flag count (viewable in the Activity tab of your user profile), and your number of reviews displayed in your profile (viewable in the Activity -> all actions -> reviews tab of your user profile). Click the "Download CSV" link on the SEDE query page, and then upload the csv file below.

Step 2: Query for Information from meta.so

Please run the linked SEDE query, again entering your Stack Overflow user ID (the same one as above). Click the "Download CSV" link on the SEDE query page, and then upload the csv file below.

Step 3: Query for Information from meta.se

Please run the linked SEDE query, this time entering your Meta Stack Exchange user ID (different from the one above, and found in the URL when you visit your profile page on meta.stackexchange.com). Click the "Download CSV" link on the SEDE query page, and then upload the csv file below. If you do not have an account, simply don't upload a file.

Step 4: See the New Visualization
